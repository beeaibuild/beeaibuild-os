# Example: Small-Scale Residential Project (Concept)

Demonstrates how BeeAiBUILD-OS handles a minimal, low-complexity urban residential request.

## Intent
Assess feasibility and basic constraints for a small-scale residential building on an urban lot.

## SIM
type: residential  
scale: small (2–3 units)  
location: urban lot  
services: existing utilities assumed  

## Tokens Triggered
REG.ZONING_RES_001  
ENG.FOUNDATION_CLASS_001  
PM.MINOR_WORKFLOW_002  

## Output (CTR Shell – structure only)
feasible: YES  
constraints: zoning height limits, lot coverage rules  
risks: minimal (standard permitting only)  
timeline: 6–9 months (band estimate)  
cost envelope: A1–A2
